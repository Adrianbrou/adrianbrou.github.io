# Portfolio Website

A modern, responsive portfolio built with React, Tailwind CSS, and Vite.

## Features
- 🎨 Modern UI with Tailwind CSS
- 🌓 Dark/Light mode toggle
- 📱 Fully responsive
- ⚡ Fast performance with Vite
- 🎭 Smooth animations
- 📄 8 pages (Home, About, Career, Education, Projects, Links, Blog, Contact)

## Getting Started

1. Install dependencies:
\`\`\`bash
npm install
\`\`\`

2. Run development server:
\`\`\`bash
npm run dev
\`\`\`

3. Build for production:
\`\`\`bash
npm run build
\`\`\`

## Tech Stack
- React 18
- React Router
- Tailwind CSS
- Framer Motion
- Lucide React Icons
- Vite
